Regex : Regular Expressions

Regex ? It is a pattern which help us in finding/searching/identifying a word/letter in a string.


The cat eat rats which are fat.

Syntax /<REGEX>/
E.g /T*/

Symbols in Regex

* => 0 or more occurances
+ => 1 or more occurances
? => Optional
. => Anything

\w => Anything which is word
\W => Anything which is not word
\s => Anything which is space
\S => Anything which is not space
\d => Anything which is digit
\D => Anything which is not digit

{minLength, maxLength} => for min and max length
[] => Group with a range

Options
g => Global
i => Case insensitive
